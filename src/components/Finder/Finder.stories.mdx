# Finder concept

The Finder will be the main screen for most apps, providing concept for:

1. Creating new content
2. Search
3. Navigating content structures
3. Bulk editing

# Finder apps

Finder apps are built using AppLayout and FinderLayout, or the corresponding building blocks.
